📋 Cum obții fișierul lastData.dat
  1. Instalează AlecaFrame dacă nu îl ai deja → alecaframe.com
  2. Pornește Warframe și lasă AlecaFrame să se sincronizeze (iconița din system tray)
  3. Deschide File Explorer și navighează la: %localappdata%\AlecaFrame
  4. Găsești fișierul lastData.dat — trage-l mai jos sau apasă pe „alege fișierul"
